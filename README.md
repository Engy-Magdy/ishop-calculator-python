# 🛒 iShop Calculator

## 📝 Description
An ishop calculator to calculate any number of items and their prices.

---

## 🧠 What I Learned
Through building this project, I focused on mastering loops and data storage:
* **Loops:** Using a `for loop` to repeat prompts dynamically based on the number of items.
* **Lists:** Storing and managing the gathered names and prices.
* **Input Handling:** Using string methods (`.lower()`/`.upper()`) and built-in functions (`sum()`) to process data.

---

## 🛠️ Technologies Used
* **Python 3.x**
